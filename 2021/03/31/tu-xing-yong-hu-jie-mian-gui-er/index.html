图形用户界面GUI（二）

No tag
Publish Date:   2021-03-31

一、Python绘图模块概述

  • Python标准库中包括下列图形绘制相关模块
    • （1）tkinter：画布绘图
    • （2）turtle：海龟绘图

二、基于tkinter的图形绘制

  • Canvas（画布）是一个长方形的区域，用于图形绘制或复杂的图形界面布局，可以在画布上绘制图形、文字，放置各种组件和框架
  • Canvas组件对象包括下列方法（绘制函数），用于绘制各种图形对象

1.创建画布对象

画布的左上角为坐标原点(0,0)，右下角为画布的大小(x,y)
c=tkinter.Canvas(parent,option=value,…)

例1：创建一个大小为200*100、背景为黄色的画布

from tkinter import *
root=Tk()
c=Canvas(root,bg='yellow',width=200,height=100)    
c.pack()    #调用组件的pack方法，调整其显示位置和大小

2.绘制矩形

在Canvas对象c上绘制线条（直线或折线）的方法如下：
id=c.create_rectangle(x0,y0,x1,y1,option,…)
（(x0,y0)代表图形左上角，(x1,y1)代表图形右下角）

例2：矩形绘制示例

from tkinter import *
root=Tk()
c=Canvas(root,bg='white',width=130,height=70)
c.pack()    #创建并显示Canvas
c.create_rectangle(10,10,60,60,fill='red')    #红色填充矩形
c.create_rectangle(70,10,120,60,fill='red',outline='blue',width=5)
#蓝色边框红色填充矩形，边框宽度为5

3.绘制椭圆

在Canvas对象c上绘制椭圆的对象方法如下：
id=c.create_oval(x0,y0,x1,y1,option,…)
（(x0,y0)代表图形左上角，(x1,y1)代表图形右下角）

例3：椭圆绘制示例

from tkinter import *
root=Tk()
c=Canvas(root,bg='white',width=280,height=70)
c.pack()
c.create_oval(10,10,60,60,fill='red')    #红色填充椭圆
c.create_oval(70,10,120,60,fill='red',outline='blue',width=5)
#红色填充蓝色边框宽度为5的椭圆
c.create_oval(130,25,180,45,dash=(4,))    #虚线椭圆
c.create_oval(190,10,270,50,dash=(4,),width=2)    #宽度为2的虚线椭圆

4.绘制圆弧

在Canvas对象c上绘制圆弧的方法如下：
id=c.create_arc(x0,y0,x1,y1,option,…)
（(x0,y0)代表图形左上角，(x1,y1)代表图形右下角）

  • 选项style用于设置圆弧的样式

例4：圆弧绘制示例

from tkinter import *
root=Tk()
c=Canvas(root,bg='white',width=250,height=70)
c.pack()
c.create_arc(10,10,60,60,style=PIESLICE)    #绘制PIESLICE样式圆弧
c.create_arc(70,10,120,60,style=CHORD)    #绘制CHORD样式圆弧
c.create_arc(130,10,180,60,style=ARC)    #绘制ARC样式圆弧
for i in range(0,360,60):    #绘制菊花瓣蓝色边框红色填充的图形
    c.create_arc(190,10,240,60,fill='red',outline='blue',start=i,extent=30)

5.绘制线条

在Canvas对象c上绘制线条（直线或折线）的方法如下：

id=c.create_line(x0,y0,x1,y1,…,xn,yn,option,…)

例5：线条绘制示例

from tkinter import *
root=Tk()
c=Canvas(root,bg='white',width=250,height=70)
c.pack()
c.create_line(10,10,60,60,arrow=BOTH,arrowshape=(3,5,4))    #双向箭头
c.create_line(70,10,95,10,120,60)    #折线
c.create_line(130,10,180,10,130,60,180,60,width=10,arrow=BOTH)    #Z字型双向箭头
c.create_line(190,10,240,10,190,60,240,60,width=10,joinstyle=MITZER)    #Z字型

6.绘制多边形

在Canvas对象c上绘制多边形的对象方法如下：

id=c.create_polygon(x0,y0,x1,y1,…,option,…)

例6：多边形绘制示例

from tkinter import *
root=Tk()
c=Canvas(root,bg='white',width=250,height=70)
c.pack()
c.create_polygon(35,10,10,60,60,60,fill='white',outline='black')    #黑边等腰三角形
c.create_polygon(70,10,120,10,120,60,fill='white',outline='black')    #黑边直角三角形
c.create_polygon(130,10,180,10,180,60,130,60)    #黑色填充的正方形
c.create_polygon(190,10,240,10,190,60,240,60,fill='white',outline='black')    
#对顶三角形

7.绘制字符串

在Canvas对象c上绘制字符串的对象方法如下：

id=c.create_text(x,y,option,…)

  • 其中，(x,y)是字符串放置的中心坐标；option为选项，包括：activefill、activestipple、anchor、disabledfill、disabledstipple、fill、font、justify、offset、state、stipple、tags、text、width。其中，text用于指定要绘制的字符串；font用于指定字体；justify用于指定对齐方式

三、基于turtle模块的海龟绘图

  • 所谓的海龟绘图，即假定一只海龟（海龟带着一只钢笔）在一个屏幕上来回移动，当它移动时会绘制直线。海龟可以沿直线移动指定的距离，也可以旋转一个指定的角度
  • 通过编写代码，可以控制海龟移动和绘图，从而绘制出图形

1.tuetle模块概述

  • （1）导入turtle模块
  • （2）创建海龟对象（turtle模块同时实现了函数模式，故也可以不创建海龟对象，直接调用函数绘图）
  • （3）设置海龟的绘图属性（画笔的属性，颜色、画线的宽度）
  • （4）控制和操作海龟绘图
    • pensize(width)/width(width) #绘制图形时的宽度
    • color(colorstring) #绘制图形时的画笔颜色和填充颜色
    • pencolor(colorstring) #绘制图形的画笔颜色
    • fillcolor(colorstring) #绘制图形的填充颜色

2.控制和操作海龟绘图

例7：使用海龟绘图绘制一个正方形

import turtle
p=turtle.Turtle()    #创建海龟对象
p.color("red")    #设置绘制时画笔的颜色
p.pensize(3)    #定义绘制时画笔的线条宽度
turtle.speed(1)    #定义绘图的速度（“slowest”或者1均表示最慢）
p.goto(0,0)    #移动海龟到坐标原点(0,0)
for i in range(4):
    p.forward(100)    #向前移动100
    p.right(90)    #向右旋转90°

例8：使用海龟绘图绘制三角形、正方形、正五边形、……、正十边形等多边形

import turtle
def draw_polygon(sides,side_len):    #绘制指定边长度的多边形
    for i in range(sides):
        turtle.forward(side_len)    #绘制边长
        turtle.left(360.0/sides)    #旋转角度
def main():
    for i in range(3,11):    
        step=50    #边长（海龟步长）为50
        draw_polygon(i,step)    #绘制多边形
if __name__=='__main__':
    main()

例9：使用海龟绘图分别绘制红、蓝、绿、黄四种颜色的圆形螺旋

import turtle
p=turtle.Turtle()
p.speed(0)    #定义绘图速度（“fastest”或者0均表示最快）
colors=["red","blue","green","yellow"]
for i in range(100):
    p.pencolor(colors[i%4])    #设置画笔颜色
    p.circle(i)
    p.left(91)

例10：科赫曲线（KOCHCURVE）的绘制

n阶科赫曲线的递归绘制算法步骤如下：
（1）基本情况（当n=0时）：绘制一条直线
（2）递归步骤（当n≥1时）：绘制一条阶数为n-1的科赫曲线，向左旋转60°，绘制第二条阶数为n-1的科赫曲线；向右旋转120°，绘制第三条阶数为n-1的科赫曲线；向左旋转60°，绘制第四条阶数为n-1的科赫曲线

import turtle
def koch(size,n):
  if n==1:
    turtle.fd(size)
  else:
    for i in [0,60,-120,60]:
      turtle.left(i)
      koch(size/3,n-1)
def main():
  turtle.setup(600,600)
  turtle.penup()
  turtle.speed(0)
  turtle.hideturtle()
  turtle.pensize(2)
  turtle.goto(-200,100)
  turtle.pendown()
  level=4
  koch(400,level)
  turtle.right(120)
  koch(400, level)
  turtle.right(120)
  koch(400, level)
  turtle.penup()
  turtle.done()
if __name__=='__main__':
    main()
